# fi-tracker

## Getting Started

Copy the .env.sample file to .env and fill in the values.

Install dependencies:

```bash
yarn install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
